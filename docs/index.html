MilliHash API (v1.0.0)

Download OpenAPI specification:

Public MilliHash REST API.

Authenticated endpoints (RapidAPI ingress) require X-RapidAPI-Key and are used for registration, owner portfolio, and API key validation.

Public verification endpoints require no authentication and support asset verification, certificate verification, and Bitcoin anchor status.

Health

RapidAPI authentication probe

Validate RapidAPI-authenticated ingress

Returns 200 when the request arrives with valid RapidAPI identity headers (post-gateway validation). Returns 401 when authentication is missing or invalid. Used exclusively for client-side key probe before local persistence (FR7-003).

Registrations

Asset hash registration (authenticated)

Register an asset hash with RFC 3161 TSA timestamp

Synchronous registration pipeline: resolve or accept SHA-256 hash, acquire TSA token, enforce global hash uniqueness, persist Registration, enqueue AnchorJob. Asset bytes are never stored. Client-supplied mode accepts hash hex only (no file upload).

Owner Portfolio

Owner registry and certificate download (authenticated)

List Registrations owned by the authenticated user

Owner-scoped, cursor-paginated registry dashboard. Fixed page size 50. Default sort: tsaTimestamp descending, registrationId ascending tie-break. Optional sort: label ascending (Unicode root collation, case-sensitive) with registrationId tie-break. Does not invoke FR6 anchor lookup per row (NFR2).

Authorizations:
RapidApiKey
query Parameters
cursor
string

Opaque signed pagination cursor from prior nextPageCursor. Omit for first page.

sort
string
Default: "tsaTimestamp"
Enum: "tsaTimestamp" "label"

Primary sort field. Default tsaTimestamp.

Public Verification

Asset, certificate, and anchor verification (unauthenticated)

Verify an asset by file or text upload

Server-computed SHA-256 (identical normalization to FR1). Exact hash lookup. No authentication. Response identical for anonymous and authenticated callers. Does not re-extract C2PA from upload — echoes stored metadata only.
